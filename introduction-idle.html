Environnement de Développement Intégré IDLE¶

Avertissement

Pour ce cours, il n’est pas recommandé d’utiliser IDLE mais plutôt le logiciel Spyder, voir Introduction à Python. Néanmoins, vous trouverez sur cette page une présentation d’IDLE au cas où vous auriez la nécessité de l’utiliser.

IDLE est l”Environnement de Développement Intégré (EDI) disponible par défaut avec Python.

IDLE comprend une fenêtre de Shell, aussi appelée console, dans laquelle il est possible d’exécuter des instructions de manière interactive. L’invite de commande est indiquée comme ceci :

>>>

Il suffit de taper une instruction puis d’appuyer sur la touche « Entrée » de votre clavier pour lancer son exécution.

On peut modifier une instruction sans tout retaper. Pour cela, il suffit de cliquer n’importe où dans l’instruction concernée et d’appuyer sur la touche « Entrée » : l’instruction est recopiée en face de l’invite de commande, et on peut la modifier comme si le texte était en cours de frappe.

Autre méthode : la combinaison de touches Alt-p (p = previous) affiche l’instruction précédente et la combinaison Alt-n (n = next) l’instruction suivante.
